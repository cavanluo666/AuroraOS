# AuroraOS

> 复杂 · 低效 · 将你的设备卡炸  
> 基于 Windows PE 的炸裂级操作系统

---

## 📌 简介

**AuroraOS** 是一个由 **13 岁初中生** 制作的、基于 Windows PE 的“操作系统”。它集成了 360 浏览器，支持运行 `.exe` 程序，界面采用远古风格，操作体验**极不友好**——专为“赤石”而设计。


---

## ✨ 特性

- 基于 Windows 10 PE 内核（版本较旧）
- 野化启动速度（慢到怀疑人生）
- 去除 Wi-Fi 自动连接工具（手动输入密码，享受复古感）
- 内置 360 浏览器（懂的都懂）
- 便携版支持 Python 跨平台运行（仅限 ZIP 包）

## 🔧 使用说明

1. **ISO 版**：使用 Rufus 或类似工具写入 U 盘，从 U 盘启动即可进入 PE 环境。
2. **便携版**：解压 ZIP 包，运行 `main.py`（需要 Python 环境）。

⚠️ **注意**：本系统仅供娱乐或技术研究，请勿在生产环境使用！
